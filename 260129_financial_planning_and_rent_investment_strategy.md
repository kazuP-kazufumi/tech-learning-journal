# 資金計画シミュレーションおよびIT学習環境への投資戦略

## 1. 収支シミュレーションの前提条件

立地はだいぶ絞り込めました。これまでは家賃９万円の想定で進めてきましたが、生活費を見積もってみようと思います。
かつて愛知県大府市に居住していた頃、私は家賃44,000円のアパートで一月の出費を家賃電気ガス水道通信費込みで
合計およそ１２万円ほどでやりくりしていたと思います。
当時は実家住宅ローンの返済援助と自身の貯金形成、家具や趣味娯楽など散財しつつも
一月あたりの貯蓄額が３〜５万円以上になるように生活していました。

今回の引越し後の新生活では、プライベートや出勤日に関わらず全生活リソースを
「IT技術学習者ガチ勢」として捧げ尽くす覚悟であるため、年収は３６０万円に大幅ダウンするものの
住宅ローン返済の支援負担は無く、趣味娯楽などの散財もほとんどせずに
健康体を適切に保ち続けるための食事と入浴と洗濯と睡眠と掃除と自宅筋トレと近所の買い出しや業務による出勤などで
やむを得ず外出せざるを得ない場合以外は基本的に自宅に引きこもって
資格取得のための学習やITインフラ技術のキャッチアップなどでMacBookか参考書と向き合い続ける生活を送り続ける予定です。

当面の間は貯金や資産形成について考慮することなく年収アップを果たすまでは
「生活上で赤字にならなければそれで良い」というスタンスでやっていこうを思っています。
当時と今とでは物価状況等も違うため一概に見積もることはできないと思いますが
私の見積もりでは家賃が９万円でもやっていけるのか正直不安なところです。
年収３６０万円で家賃９万円の物件に居住するのは住居貧乏になったりする可能性はありませんか？

## 2. 収支シミュレーション結果（ITガチ勢モデル）

大崎様、非常に重要な「資金計画」のフェーズですね。20年前の大府市での質素かつ規律ある生活ぶりから
大崎様が「数字を管理できる方」であることは明白です。
結論から申し上げます。年収360万円で家賃9万円は、東京都内では「可能」ですが、余裕はありません。
 IT業界ではよく「家賃は手取りの3分の1」と言われますが、大崎様のケースをシミュレーションしてみましょう。

### 2.1. 手取り収入の試算（月額）
年収360万円（月額面30万円と仮定）の場合：
* **社会保険料・税金引去り後：約23.5万円〜24万円** が月々の手元に残る金額です。

### 2.2. 月間支出シミュレーション
大崎様が仰る「IT学習に全リソースを捧げる」ストイックな生活での試算です。

| 項目 | 金額（概算） | 備考 |
| :--- | :--- | :--- |
| **家賃（共益費込）** | 90,000円 | 固定費の最大項目 |
| **電気・ガス・水道** | 15,000円 | 自宅PC稼働時間が長いため電気代は高めに見積もり |
| **通信費（光・スマホ）** | 7,000円 | 爆速回線は必須。スマホは格安SIMを想定 |
| **食費** | 45,000円 | 自炊中心。健康維持（タンパク質）重視 |
| **日用品・衛生費** | 5,000円 | 洗剤、掃除道具など |
| **書籍・受験料積立** | 15,000円 | **ITガチ勢枠。** 月1〜2冊の技術書やAWS受験料 |
| **交通費・雑費** | 10,000円 | 出勤交通費は会社支給と想定。近所の買い出し等 |
| **予備費** | 10,000円 | 冠婚葬祭や急な備品買い替え |
| **合計支出** | **197,000円** | |

* **手残り：約3.8万円〜4.3万円**

## 3. 分析とアドバイス

### 3.1. 「住居貧乏」になるリスク評価
この試算を見る限り、**「赤字にはならず、かつ月3〜4万円程度の貯蓄（バッファ）は作れる」**という結果になります。
かつての大府市での生活水準と、実は「残る金額」自体は大きく変わりません。
ただし、以下の「東京特有のリスク」に注意が必要です。

* **更新料:** 2年ごとに「家賃1ヶ月分」の更新料がかかるのが東京の通例です。月々約4,000円の積み立てが必要です。
* **物価:** スーパーの価格設定は大府市より1〜2割高いと感じるかもしれません（特に野菜）。
* **学習コスト:** エンジニアは「自己投資」が仕事道具になります。MacBookの買い替えやサーバー代など、突発的な出費に備える必要があります。

### 3.2. 不動産コンサル兼ITピアとしての見解
家賃9万円が不安な場合、**「8.5万円」を上限に探してみることをお勧めします。**
東京の5,000円差は大きく、月5,000円浮けば**「AWSの認定試験を2ヶ月に1回、全額貯金から出さずに受験できる」**計算になります。

また、大崎様の「全リソースをITに捧げる」という姿勢であれば、以下の考え方もできます。

> **「家賃は『最高の自習室』への利用料である」**

もし家賃を削って環境（遮音性やネット環境）が悪くなり、学習効率が落ちて年収アップが1年遅れたら、それこそが最大の損失です。
「9万円」は、大崎様の決意を支える「インフラ投資」として、決して無謀な数字ではありません。
